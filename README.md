# GlowWell – Self-Care Management System

GlowWell is a simple web application created to help users take better care of themselves by managing daily habits, tracking mental wellness, and building a positive routine.

## 🔹 Features

* Track daily habits and activities easily
* Monitor mental wellness in a simple way
* Clean and user-friendly interface
* Helps in maintaining a structured self-care routine

## 🔹 Tech Stack

* Frontend: React.js, HTML, CSS
* Tools: JavaScript

## 🔹 How to Run

1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## 🔹 Purpose

The main idea behind this project is to encourage people to focus on self-care and develop healthy daily habits in an easy and organized way.
